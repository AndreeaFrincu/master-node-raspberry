Overview

Smart Building Collector device: RaspberryPi-02

This node presents a classification of data collection methods to monitor the thermal comfort and luminous intensity of the building, thus promissing good indoor environment for the building's occupants.

Data Center Collector device: RaspberryPi-03

This node focuses on monitoring supported elements such as data center temperature and humidity to be up to date with heating and ventilation systems' health, and air quality, meaning measuring the levels of CO2 to prevent fires.
